# 飞书群组机器人配置指南

Cloud Mail 支持将收到的新邮件实时推送到你的飞书群组。以下是配置步骤。

## 第一步：在飞书中添加自定义机器人

1. 打开飞书桌面端，进入你想要接收邮件通知的**群组**。
2. 点击群组右上角的「设置」（齿轮图标）。
3. 选择「群机器人」 -> 「添加机器人」。
4. 在机器人列表中找到并选择「自定义机器人」，点击「添加」。
5. 为机器人设置一个名称（如：Cloud Mail 通知）和头像，然后点击「添加」。
6. **复制 Webhook 地址**。这是机器人接收消息的唯一凭证，请妥善保管。
   - *注意：当前版本暂未要求开启“安全设置”（加签、自定义词），如果在飞书中开启了安全设置，请确保包含了“邮件”之类的关键词或暂时关闭加签功能。*

## 第二步：在 Cloud Mail 中配置 Webhook

1. 使用管理员账号登录你的 Cloud Mail 前台网站（如：`https://mail.你的域名.com`）。
2. 在左侧菜单中点击「管理」 -> 「系统设置 (System Settings)」。
3. 找到「邮件推送 (Email Push)」卡片。
4. 点击「飞书机器人 (Feishu Bot)」右侧的**齿轮图标**。
5. 在弹出的输入框中，粘贴你刚刚在飞书复制的 **Webhook 地址**。
6. 打开左下角的**启用开关**（变成开启状态）。
7. 点击「保存 (Save)」。

## 第三步：测试推送

1. 使用任意外部邮箱（如 Gmail、QQ 邮箱等）向你在 Cloud Mail 已经绑定的邮箱地址发送一封测试邮件。
2. 等待几秒钟，你的飞书群组应该会收到一条卡片消息，包含：
   - 邮件主题
   - 发件人名字和邮箱
   - 收件人名字和邮箱
   - 邮件内容的摘要预览
   - 一个「查看完整邮件」的按钮

点击「查看完整邮件」按钮，你将被引导至 Cloud Mail 的安全链接直接查看该封邮件（由于安全性，部分环境可能需要先登录）。

## 常见问题

**Q: 为什么飞书没有收到通知？**
A: 请检查以下几点：
1. 系统设置中的「飞书机器人」开关是否已启用。
2. Webhook 地址是否填写正确，没有多余的空格。
3. 飞书机器人的「安全设置」中是否设置了“自定义词”或“签名（加签）”。**如果是，请在飞书中将其关闭**，或者确保你的邮件主题/内容中包含飞书要求的“自定义词”。目前代码默认不计算签名。

**Q: 飞书推送是否支持附件预览？**
A: 目前的飞书消息推送仅提供文本摘要预览和邮件链接。如果邮件有附件，你需要点击「查看完整邮件」进入 Cloud Mail 系统中进行下载和查看。
