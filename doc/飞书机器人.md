# 飞书应用机器人配置指南

Cloud Mail 支持将收到的新邮件实时推送到你的飞书群组或私聊界面，采用了更安全的飞书官方开发者开放 API。

## 第一步：创建飞书企业自建应用

你已经拥有了 APP ID 和 APP Secret，无需再次创建，如果你需要更改或重新配置：
1. 访问 [飞书开发者后台](https://open.feishu.cn/app)
2. 创建一个「企业自建应用」
3. 进入应用的配置界面，**复制你的 App ID 和 App Secret**
4. 在左侧菜单找到「添加特色能力」->「机器人」，开启机器人能力
5. (重要) 给应用**发布版本**并请求通过审核（或者如果你有自审权限直接上线），只有上线的应用才能使用 `tenant_access_token` 调库发消息。

## 第二步：获取 Receive ID（群聊 ID 或 用户 ID）

要让机器人知道将消息发给谁，需要填入目标 ID：
1. 如果要发送到**群组**，这是 `chat_id`。最简单的获取方法是将机器人添加到该飞书群里后，通过飞书提供的[消息群组调试工具](https://open.feishu.cn/tool/im)查询该群的群 ID (`oc_xxxx...`)。
2. 记录这串 ID 以备使用。

## 第三步：在 Cloud Mail 中配置

1. 使用管理员账号登录你的 Cloud Mail 前台网站（如：`https://mail.你的域名.com`）。
2. 在左侧菜单中点击「管理 (Admin)」 -> 「系统设置 (System Settings)」。
3. 找到「邮件推送 (Email Push)」卡片。
4. 点击「飞书机器人 (Feishu Bot)」右侧的 **齿轮图标**。
5. 在弹出的输入框中分别填入你获得的配置信息：
   - **App ID**: 例如 `cli_a9224b8ae4f8dcee`
   - **App Secret**: 例如 `9Q5GDMoDj0GTMixW9qonCd5AOdVzayGs`
   - **Receive ID (Chat ID)**: 你的目标聊天对象 ID，如 `oc_...` 
6. 打开左下角的 **启用开关**（变成开启状态）。
7. 点击「保存 (Save)」。

## 第四步：发送并测试你的设置

1. 使用任意外部邮箱（如 Gmail、QQ 邮箱等）向你在 Cloud Mail 已经绑定的邮箱地址发送一封测试邮件。
2. 等待几秒钟，你的飞书客户端中该群组应该会收到一条**富文本消息 (Post)**，包含：
   - 具有格式的主题、发件人和收件人
   - 邮件普通文本形式的前 200 个字摘要
   - 带有可点击图标的【查看完整邮件】外链

如果你没有收到信息，可以按 F12 并在 Cloud Mail Workers 项目的本地日志查找控制台是否有 API `tenant_access_token` 读取错误或是 `im/v1/messages` 接口反馈的消息发送失败日志来对症处理。
